# Sales Performance Dashboard (Excel)

## Project Overview

This project focuses on analyzing sales data and building an interactive **Sales Performance Dashboard** using Microsoft Excel. The goal of the project is to understand business performance by analyzing sales across different **categories, regions, and time periods**.

The dataset was cleaned and prepared before creating pivot tables and visualizations to highlight key business insights.

---

## Dataset

The dataset used for this project is the **Superstore Sales Dataset**, which contains transactional sales data including:

* Order Date
* Sales
* Quantity
* Region
* Category
* Sub-Category
* Customer Segment
* Profit

The dataset contains approximately **10,000 records**, making it suitable for practicing real-world data analysis.

---

## Data Preparation

Before building the dashboard, the dataset was cleaned and organized by:

* Removing duplicate records
* Handling missing values
* Converting date columns into proper date format
* Creating additional fields such as **Order Year** and **Order Month** for time-based analysis
* Organizing the dataset for pivot table analysis

---

## Pivot Table Analysis

Several pivot tables were created to analyze sales performance from different perspectives:

* **Sales by Category** – Identifies which product categories generate the highest revenue
* **Sales by Region** – Shows regional performance and market strength
* **Monthly Sales Trend** – Analyzes sales patterns over time
* **Sales by Sub-Category** – Provides deeper insight into product-level performance
* **Quantity Sold by Category** – Compares sales volume across product categories

---

## Dashboard Features

The Excel dashboard provides a clear and interactive visualization of sales performance.

Key features include:

* Multiple charts for quick business insights
* Interactive **slicers** for filtering data by category, region, and year
* Clean and organized layout for easy interpretation
* Dynamic pivot charts connected to pivot tables

## Key Insights

The dashboard helps identify:

* High-performing product categories
* Regional sales performance
* Sales trends across months and years
* Product demand based on quantity sold

These insights can help businesses make better **data-driven decisions**.
